## Installation

1. Download the extension from **[here](https://github.com/istoocold/RILIM5/archive/refs/heads/main.zip)**  
2. Extract the ZIP file  
3. Open Chrome and go to `chrome://extensions/`  
4. Enable Developer Mode  
5. Click Load Unpacked  
6. Select the extracted RILIM5 folder that you downloaded in Step 1  
7. Done
